# CosmicJS Docs

This is build with [VuePress v1+](https://v1.vuepress.vuejs.org/).

## Local Setup

To setup the local environment:

### Prerequisites

- [Node.js v8+](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/en/) (Optional, but recommended)

### Development

```
$ yarn run dev
```

### Build

```
$ yarn build
```
